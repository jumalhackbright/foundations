Note: This resources uses an online tool called "Nitrous", which asks for a credit card in order to use their service for free, for 7 days. Candidates who cannot enter a credit card can instead use a tool called, "Koding - Solo Edition", available at www.koding.com/features/solo, which is very similar to Nitrous. The drawback is that it is slightly more complicated to use, but a mentor should be able to help you match the contents of this tutorial with Koding.
